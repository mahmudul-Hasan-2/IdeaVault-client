# 🚀 IdeaVault - Modern Idea Sharing Platform

🔗 **Live Demo:** [https://ideavault-client-psi.vercel.app/](https://ideavault-client-psi.vercel.app/)

---

## ✨ Overview

IdeaVault is a modern idea-sharing platform where users can create, explore, and interact with ideas in a smooth and user-friendly interface.

---

## ⚡ Features

- 🧠 **Add Ideas**
  Users can easily share their ideas anytime through a simple and intuitive form.

- 🔍 **Smart Idea Discovery**
  Multiple ways to search and explore ideas based on user needs.

- 📄 **Idea Details Page**
  Each idea has a dedicated page with full details and interactions.

- 🧾 **User Dashboard Pages**
  - Add Idea
  - My Ideas
  - My Interactions

- 💬 **Comment System**
  Users can discuss and interact through comments on ideas.

- 🎠 **Modern UI Banner Carousel**
  A responsive carousel added to the homepage for better UX.

---

## 🛠️ Tech Stack

- Next.js / React
- Node.js / Express
- MongoDB
- JWT Authentication
- Vercel (Deployment)

---

## 📦 Installation & Setup

```bash
# Clone repository
git clone https://github.com/your-username/ideavault.git

# Install dependencies
npm install

# Run development server
npm run dev
```
